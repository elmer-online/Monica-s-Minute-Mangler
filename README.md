## Installation

### Requirements
- Python 3.12+
- PySide6

### Setup

```bash
# Clone the repository
git clone https://github.com/elmer-online/Monica-s-Minute-Mangler.git
cd Monica-s-Minute-Mangler

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (Mac/Linux)
source venv/bin/activate

# Install dependencies
pip install PySide6

# Run
python main.py
```

## Usage

1. **Set Time**: Type directly in the display (e.g., "5" for 5 min, "2:30" for 2:30)
2. **Add Time**: Click the heart to add 5 minutes
3. **Start**: Press Enter or click Start
4. **Control**: Use Pause/Resume and Stop buttons
